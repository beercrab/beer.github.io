安全软件列表

我们应尽量避免依赖第三方软件，若确有必要使用，则优先考虑那些用户群体广泛、市场认可度高的解决方案,以下是我收录并分享了一些我正在使用的优质软件，满足可靠、可用、安全特性。

以下这些软件可以构建一个完整的私人网络，甚至可以作为整个互联网的分支

系统软件

适用电脑及服务器

  • Debian/Linux: 最稳定的Linux发行版.不需要商业授权，适用于服务器
  • Windows 11: 主流的电脑桌面系统，适合办公、娱乐、生产

应用软件

按标准，软件仅分系统软件，及运行在系统软件之上的内置应用软件和第三方应用软件。

桌面应用

  • WPS: 国产主流的跨平台办公软件，有安全团队保障
  • Xmind: 国产主流的跨平台思维导图软件
  • 火狐浏览器: 唯一不使用chromium内核且注重安全隐私的跨平台浏览器
  • PowerToys: 微软官方的Windows系统增强工具
  • Wireshark: 主流的跨平台网络分析工具
  • Github Desktop: Github桌面版，友好的视图界面，非常方便
  • 向日葵远程: 国产远控软件，有安全团队保障
  • VitrualBox： 开源的桌面虚拟机软件，对Linux系统兼容性强
  • VMware Workstation Pro： 稳定高效的主流桌面虚拟机软件

网络服务

  • iPerf: 开源网络带宽测量工具
  • dPerf: 国产开源网络带宽测量工具
  • TigerVNC: 开源免费的VNC远程软件
  • 花生壳内网穿透: 国产内网穿透软件，有安全团队保障
  • APACHE HTTP SERVER: 凡是需要上互联网访问网站都需要它
  • Apache ECharts: 一个国产基于JavaScript的开源可视化图表库，对于网站内容的填充起到很棒的作用
  • Nginx: 和上面的功能差不多，但是性能相对更高
  • WordPress : 开源免费的建站软件
  • MariaDB: 开源免费的数据库软件
  • MySQL: 开源免费的数据库软件
  • XAMPP: 可以一键安装Apache+MariaDB+PHP
  • phpmyadmin: 跨平台MySQL/MariaDB数据库管理工具
  • php: 建站首选环境
  • Node.js: 可以在本机电脑搭建基于Javascript静态网站的运行环境程序
  • FRRouting: 开源动态路由协议套件，适合用来学习掌握网络路由技术，也可以将Linux打造成路由器

MySQL和MariaDB两个几乎一样，只不过MySQL已经被Oracle收购了，但是依然有开源社区版本

安全隐私

  • AdGuard Home: 保护家庭网络一款全网广告拦截与反跟踪软件
  • AdGuard： 网站广告过滤及防跟踪浏览器插件
  • 隐私獾: 防止第三方跟踪网站浏览行为浏览器插件
  • snort: 适用Linux开源入侵检测系统
  • netfilter: 强大的Linux开源防火墙系统核心框架
